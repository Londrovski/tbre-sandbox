# Simulation Developer

Owns the Unity simulation used to test the control systems. Built by Ben Rall; currently models the ASD-DV car (the only car the AI team has run to date).

## Responsibilities
- **Run the sim** — Manage simulation setup; help other members use the sim where needed.
  - context: Sim originally built by Ben Rall
  - owns: TBReSim repo (gitlab tbre-ai/tbresim)
  - delivers: A working sim others can use
- **Fix hard bugs** — Fix key bugs other members can't solve.
  - owns: Sim bug triage
  - delivers: A stable simulation
- **Keep it realistic** — Keep the sim representative of the real test vehicle.
  - context: Vehicle params from VD / Mounting
  - delivers: Sim matched to TBRe27 (height, weight, tyres)

## Key interfaces
- Mounting Engineer (sensor plate location in GA/sim)
- Control / MPC Developer

## Requirements
- Comfortable with Unity / C#
- Interested in simulation and vehicle modelling
- ~2-3 hrs/week through the year
