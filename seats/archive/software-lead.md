# Software Lead

Owns the software domain: technical direction across all software seats, plus onboarding, repo hygiene and recruitment for software.

## Responsibilities
- Set technical direction and standards across the software seats (SLAM, perception, control, path-planning, sim)
- Recruitment for software roles
- Onboard new members to GitLab and Teams
- GitLab management – repository structure, access, merge/review standards
- Unblock software seats and own cross-seat integration

## Goals
- Every software seat owned and onboarded by start of season
