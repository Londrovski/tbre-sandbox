# TBRe Team Structure (sandbox)

Public sandbox for mapping out TBRe team roles. Starts with the **AI team** as the template.

- **Live role map:** GitHub Pages (enable under Settings -> Pages -> Source: GitHub Actions). The URL appears there after the first deploy.
- **`index.html`** - interactive role map (reads `seats.json`).
- **`seats.json`** - the role data the map renders.
- **`roles/`** - one markdown card per role (purpose, responsibilities, interfaces, goals).

Each role is a *seat*: one accountable owner, a few core responsibilities, ~2-3 hrs/week. One person can hold several seats.
